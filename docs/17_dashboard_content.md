## 3. 통계 요약

프로젝트 상태별 집계. 4개 수치 카드.

| 카드 | 레이블 | 값 |
|---|---|---|
| 통계 1 | 전체 | 전체 프로젝트 수 |
| 통계 2 | 완성 | 완성 상태 수 |
| 통계 3 | 임시저장 | 임시저장 상태 수 |
| 통계 4 | 처리중 | 처리중 상태 수 |

프로젝트가 없을 때 (빈 상태): 통계 행 숨김.

## 4. 프로젝트 목록 영역

### 4-1. 상태 필터 탭

| 탭 | 레이블 | 표시 조건 |
|---|---|---|
| 탭 1 | 전체 | 항상 표시 |
| 탭 2 | 완성 | 항상 표시 |
| 탭 3 | 임시저장 | 항상 표시 |
| 탭 4 | 처리중 | 항상 표시 |
| 탭 5 | 실패 | 항상 표시 |

기본 선택: 전체.

### 4-2. 정렬 선택

| 옵션 | 레이블 | 정렬 기준 |
|---|---|---|
| 기본 | 최신순 | 마지막 수정일 내림차순 |
| 옵션 2 | 오래된 순 | 마지막 수정일 오름차순 |
| 옵션 3 | 이름순 | 논문 제목 가나다순 |

### 4-3. 프로젝트 카드 텍스트

| 요소 | 내용 | 말줄임 |
|---|---|---|
| 논문 제목 | S6에서 추출한 논문 제목. 없으면 파일명 사용. | 2줄 |
| 파일명 | 업로드된 원본 파일명 | 1줄 |
| 날짜 | YYYY.MM.DD | 마지막 수정일 기준 |
| 카드 수 | N장 | — |
| 상태 배지 | 완성 · 임시저장 · 처리중 · 실패 | — |

### 4-4. 상태별 액션 버튼

| 상태 | 버튼 1 | 버튼 2 | 비고 |
|---|---|---|---|
| 완성 | 수정하기 | 다운로드 | 다운로드 → 에디터 진입 + 내보내기 모달 자동 오픈 |
| 임시저장 | 이어하기 | — | 에디터 진입 |
| 처리중 | (없음) | — | 카드 전체 비활성 |
| 실패 | 재시도 | — | 실패 지점부터 파이프라인 재실행 |

### 4-5. 처리중 카드 오버레이 텍스트

카드 썸네일 위에 현재 단계 표시.

| 파이프라인 단계 | 오버레이 텍스트 |
|---|---|
| S1 진행 중 | 텍스트 추출 중... |
| S2 진행 중 | 섹션 분류 중... |
| S6 진행 중 | 핵심 내용 분석 중... |
| S7 진행 중 | 카드 생성 중... |
| S8 진행 중 | 저장 중... |

### 4-6. 실패 카드 배지 텍스트

| 실패 단계 | 배지 | 툴팁 (hover 시) |
|---|---|---|
| S1 실패 | 텍스트 추출 실패 | 스캔 PDF는 처리할 수 없습니다. |
| S2 실패 | 섹션 분류 실패 | 문서 구조를 파악하지 못했습니다. |
| S6 실패 | 분석 실패 | 핵심 내용을 추출하지 못했습니다. |
| S7 실패 | 카드 생성 실패 | 카드를 생성하지 못했습니다. |
