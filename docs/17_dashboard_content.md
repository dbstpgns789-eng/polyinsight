## 4. 프로젝트 목록 영역

### 4-1. 검색

텍스트 입력으로 논문 제목 또는 파일명을 검색한다.

| 요소 | 텍스트 |
|---|---|
| 입력 플레이스홀더 | 제목이나 파일명으로 검색 |

검색어가 있을 때 필터·정렬은 검색 결과 안에서 동작한다.

### 4-2. 상태 필터 탭

| 탭 | 레이블 | 표시 조건 |
|---|---|---|
| 탭 1 | 전체 | 항상 표시 |
| 탭 2 | 완성 | 항상 표시 |
| 탭 3 | 임시저장 | 항상 표시 |
| 탭 4 | 처리중 | 항상 표시 |
| 탭 5 | 실패 | 항상 표시 |

기본 선택: 전체.

### 4-3. 정렬 선택

| 옵션 | 레이블 | 정렬 기준 |
|---|---|---|
| 기본 | 최신순 | 마지막 수정일 내림차순 |
| 옵션 2 | 오래된 순 | 마지막 수정일 오름차순 |
| 옵션 3 | 이름순 | 논문 제목 가나다순 |

### 4-4. 현재 조회 결과 수

필터·검색 결과에 따라 동적으로 갱신한다.

| 상황 | 표시 텍스트 |
|---|---|
| 전체 탭 | 총 N개 카드뉴스 |
| 상태 탭 선택 | 완성 N개 / 임시저장 N개 / 처리중 N개 / 실패 N개 |
| 검색어 있음 | "검색어" 결과 N개 |

### 4-5. 프로젝트 카드 썸네일

각 프로젝트 카드에는 썸네일 영역이 있다.

| 상태 | 썸네일에 표시되는 것 |
|---|---|
| 완성 | 생성된 카드뉴스의 첫 번째 장 이미지 |
| 임시저장 | 생성된 카드뉴스의 첫 번째 장 이미지 |
| 처리중 | 진행 단계 텍스트 + 진행률 (§4-7 참조) |
| 실패 | 실패 배지 텍스트 (§4-8 참조) |

썸네일 위에 표시되는 배지 두 가지:

| 배지 | 내용 | 예시 |
|---|---|---|
| 비율 배지 | 카드 출력 비율 | 1:1 |
| 장수 배지 | 생성된 카드 수 | 5장 |

### 4-6. 프로젝트 카드 텍스트 정보

| 요소 | 내용 | 말줄임 |
|---|---|---|
| 논문 제목 | S6에서 추출한 논문 제목. 없으면 파일명 사용. | 2줄 |
| 파일명 | 업로드된 원본 파일명 | 1줄 |
| 날짜 | YYYY.MM.DD | 마지막 수정일 기준 |
| 상태 배지 | 완성 · 임시저장 · 처리중 · 실패 | — |

### 4-7. 상태별 액션

| 상태 | 액션 1 | 액션 2 | 액션 3 | 비고 |
|---|---|---|---|---|
| 완성 | 수정하기 | 다운로드 | 삭제 | 다운로드 → 에디터 진입 + 내보내기 모달 자동 오픈 |
| 임시저장 | 이어하기 | 삭제 | — | 이어하기 → 에디터 진입 |
| 처리중 | (없음) | — | — | 카드 전체 비활성 |
| 실패 | 재시도 | 삭제 | — | 재시도 → 실패 지점부터 파이프라인 재실행 |

삭제 동작: 확인 없이 즉시 삭제하지 않는다. "삭제하시겠습니까?" 확인 후 처리.

### 4-8. 처리중 카드 진행 단계 텍스트

| 파이프라인 단계 | 표시 텍스트 |
|---|---|
| S1 진행 중 | 텍스트 추출 중... |
| S2 진행 중 | 섹션 분류 중... |
| S6 진행 중 | 핵심 내용 분석 중... |
| S7 진행 중 | 카드 생성 중... |
| S8 진행 중 | 저장 중... |

### 4-9. 실패 카드 배지 및 설명

| 실패 단계 | 배지 텍스트 | 부연 설명 (카드 내 표시) |
|---|---|---|
| S1 실패 | 분석 실패 | 스캔 PDF는 처리할 수 없습니다. |
| S2 실패 | 분석 실패 | 문서 구조를 파악하지 못했습니다. |
| S6 실패 | 분석 실패 | 핵심 내용을 추출하지 못했습니다. |
| S7 실패 | 분석 실패 | 카드를 생성하지 못했습니다. |

## 5. 빈 상태 (프로젝트 0개)

첫 방문이거나 모든 프로젝트를 삭제한 경우.
통계 행과 필터·검색 영역은 숨긴다.

| 요소 | 텍스트 |
|---|---|
| 안내 제목 | 첫 논문을 업로드하세요 |
| 안내 설명 | PDF를 올리면 카드뉴스를 자동으로 만들어 드립니다. |
| CTA 버튼 | 새 카드뉴스 만들기 |

## 6. 필터·검색 결과 없음 상태

필터 탭 선택 또는 검색 후 결과가 없을 때.

| 상황 | 안내 텍스트 |
|---|---|
| 완성 탭 결과 없음 | 완성된 카드뉴스가 없습니다. |
| 임시저장 탭 결과 없음 | 임시저장된 프로젝트가 없습니다. |
| 처리중 탭 결과 없음 | 처리 중인 프로젝트가 없습니다. |
| 실패 탭 결과 없음 | 실패한 프로젝트가 없습니다. |
| 검색 결과 없음 | "검색어"와 일치하는 카드뉴스가 없습니다. |
