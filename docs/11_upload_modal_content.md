# 업로드 모달 — 콘텐츠 명세

> 두 번째 화면. 랜딩페이지에서 "무료로 시작하기" 클릭 시 진입.
> 상태 3개: IDLE → FILE_SELECTED → PROCESSING (→ DONE | ERROR)

---

## 상태 1 — IDLE (파일 선택 전)

### 모달 헤더
- 제목: `카드뉴스 생성`
- 닫기 버튼: 있음

### 파일 업로드 영역
- 안내 문구: `PDF 파일을 드래그하거나 클릭해서 선택`
- 제약 조건: `최대 50MB · PDF 형식만 허용`
- 상호작용: 클릭 시 파일 탐색기 열림 / 파일 드래그&드롭 가능

### 카드 수 선택
- 레이블: `카드 수`
- 범위: 3장 ~ 15장 (기본값: 5장)
- 양 끝 레이블: `3장 (간결)` / `15장 (상세)`
- 현재 값 표시: `N장`

### 실행 버튼
- 텍스트: `카드뉴스 생성 시작`
- 상태: 비활성 (파일 미선택 시)

---

## 상태 2 — FILE_SELECTED (파일 선택 후)

IDLE 상태에 추가로:

### 선택된 파일 정보
- 파일 이름: `{filename}.pdf`
- 파일 크기: `{size} MB`
- 제거 버튼: 있음 (클릭 시 IDLE로 복귀)

### 실행 버튼
- 텍스트: `카드뉴스 생성 시작`
- 상태: 활성

### 에러 (유효하지 않은 파일 선택 시)
- PDF 아닌 파일: `PDF 파일만 업로드할 수 있습니다.`
- 용량 초과: `파일 크기가 50MB를 초과합니다. ({실제크기}MB)`

---

## 상태 3 — PROCESSING (처리 중)

### 모달 헤더
- 제목: `카드뉴스 생성`
- 닫기 버튼: 없음 (처리 중 강제 닫기 불가)

### 진행 상태 표시
- 상단 레이블: `파이프라인 진행 중`
- 진행률: `{N}%`
- 진행 바: 0% → 100%

### 파이프라인 단계 (순서대로)
| 단계 | 이름 | 설명 |
|------|------|------|
| 1 | 텍스트 추출 | PDF 파싱 및 섹션 분리 |
| 2 | AI 카드뉴스 생성 | 내용 분석 및 구조화 |
| 3 | PNG 렌더링 | 카드 이미지 생성 |
| 4 | 패키징 | 파일 저장 및 마무리 |

각 단계 상태: 대기 중 / 진행 중 / 완료

---

## 상태 3a — DONE (처리 완료)

### 실행 버튼
- 텍스트: `카드 에디터 열기 →`
- 클릭 시: 모달 닫힘 + `/editor/{jobId}` 이동

---

## 상태 3b — ERROR (처리 실패)

### 에러 메시지
- 제목: `처리 중 오류가 발생했습니다.`
- 안내: `PDF 파일을 확인하고 다시 시도해주세요.`

### 버튼 (2개)
- `닫기` — 모달 종료
- `다시 시도` — 처음부터 재시작

---

## 제약 조건 요약

| 항목 | 값 |
|------|----|
| 허용 파일 형식 | PDF |
| 최대 파일 크기 | 50MB |
| 카드 수 최소 | 3장 |
| 카드 수 최대 | 15장 |
| 카드 수 기본값 | 5장 |
| 처리 중 닫기 | 불가 |
