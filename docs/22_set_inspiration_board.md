## 0. 핵심 발견 — 참조들은 전부 같은 뼈대, 다른 피부

수집한 3세트(에디토리얼 라이트 / 다크 네온 / 코퍼릿 리포트)는 **구조 DNA가 동일**하다:
`키커 → 거대 헤드라인(+강조어) → 본문 → 번호 목록 → 핵심 수치 → 이미지 슬롯 → 클로징`.
이건 우리 **8뼈대 그대로**다. 차이는 **피부(색 토큰·장식·밀도·타이포 스케일)**뿐.
→ "N세트 = N개 빌드"가 아니라 **"N개 스킨 정의 + 공유 장식 컴포넌트"**. (skin/skeleton 분리의 보상)

색은 fidelity와 무관(장식). **시각 풍부함은 우리가 오히려 유리** — 장식(글로우·도형) 자유 + **논문 자체 figure**(경쟁자 불가). 단 AI생성/스톡-as-사실은 금지.

- **도메인 핏:** 바이오·화학·정책·일반(비타민C 논문에 맞음).
- **우리 매핑:** ≈ `EDITORIAL_LIGHT_SET`. 갭 = 여백·타이포대비 더, 디바이더 표준화, body `*em*` 지원.

- **타이포:** 네이비 볼드 헤드라인 + 강조어 블루.
- **시그니처:** **번호(01/02/03) 블루**, **도넛 차트**(2.5% 중앙), 점선 키커(—ANNUAL REPORT SERIES—), ○○○ 점, 알약 배지/CTA.
