# Problem Definition
> PolyInsight v2.0 | 2025-05-05

---

## 1. 문제 상황

한국생산기술연구원(KITECH) 연구자들은
논문 발표 후 웹진 게재용 카드뉴스를 제작해야 한다.

### 이해관계자별 Pain Point

| 이해관계자 | Pain Point |
|---|---|
| 논문 저자(박사) | 카드뉴스 제작 전문가가 아님. 연구에 집중하고 싶음. |
| 디자인팀 | 논문을 이해하는 데 시간 비용 과다. 도메인 지식 부족. |
| 기관(KITECH) | 카드뉴스 품질이 연구자 역량에 의존. 일관성 없음. |

### 핵심 병목

```
논문 저자    →  디자인팀에 외주 의뢰
디자인팀     →  논문 이해에 수일 소요
디자인팀     →  카드뉴스 초안 작성
저자         →  검토 및 수정 요청
디자인팀     →  재작업
→ 총 소요: 1~2주, 수차례 피드백 반복
```

---

## 2. 우리의 해결 접근

AI가 논문을 읽고 카드뉴스 초안을 자동 생성.
인간은 수치·사실 검증과 이미지 업로드만 담당.

### AI가 할 수 있는 것

- 논문 텍스트 구조 파악 및 섹션 분류
- 핵심 주장과 수치 추출
- 대중이 이해할 수 있는 언어로 재구성
- 카드뉴스 레이아웃 자동 선택

### 반드시 사람이 해야 하는 것

| 항목 | 이유 |
|---|---|
| 수치·사실 검증 | LLM hallucination 위험. 기관 신뢰 문제. |
| 연구진 사진 업로드 | 개인정보, 저작권, 공신력 |
| 웹진 월호·번호 입력 | 논문 밖의 메타데이터 |
| 최종 승인 | 공식 홍보물 책임 소재 |

---

## 3. 품질 정의

"웹진에 출판 가능한 카드뉴스"란:

- 수치 오류 0건
- KITECH 브랜드 가이드 준수
- 일반인이 30초 내 핵심 이해 가능
- 연구자가 "내 연구가 제대로 표현됐다"고 동의

---

## 4. 성공 지표

| 지표 | 목표 |
|---|---|
| 카드뉴스 생성 시간 | 기존 1~2주 → 1시간 이내 |
| 첫 생성 승인율 | MVP: 25% → v2: 50% → v3: 70% |
| 수치 오류율 | 0% (사람 검증 게이트 통과 후) |
| 사용자 만족도 | "기존보다 빠르다" 90% 이상 |
