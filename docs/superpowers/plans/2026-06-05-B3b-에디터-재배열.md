# B-3b — 에디터 카드 재배열 + 신규 뼈대 라벨 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans. Steps use checkbox (`- [ ]`) syntax.

**Goal:** AI가 고른 카드 순서를 사용자가 ▲▼ 버튼으로 재배열하고, 에디터의 카드 라벨이 신규 8뼈대를 올바로 표시하게 한다.

**Architecture:** 프론트 전용. 에디터 페이지(`editor/[jobId]/page.tsx`)에 `handleReorderCard`(배열 swap + card_num 재번호 + 자동저장) 추가 → `LeftPanel`에 `onReorderCard` 전달, 썸네일에 ▲▼ 버튼. 라벨 맵 3곳(LeftPanel·FactDrawer·MidCanvas)에 신규 8키 추가. 데이터 모델·백엔드 불변.

**Tech Stack:** Next 16 · React 19 · TS. 드래그 라이브러리 없음(▲▼ 버튼 — 카드 3~7장이라 충분, 캔버스-없는 철학 정합).

**검증:** 브라우저 — `/editor/demo`에서 라벨이 신규 뼈대명으로 뜨고, ▲▼로 순서가 바뀌며 card_num이 재번호되고 미리보기/순서가 따라오는지. (프론트 단위 테스트 인프라는 순수 로직만 — 재배열은 시각/상호작용이라 브라우저 검증.)

**비목표(후속):** RightPanel 이미지 슬롯 설정(신규 뼈대 cover_v2/feature 이미지 업로드 UI), 드래그 앤 드롭, 카드 추가/삭제, 카드별 뼈대 교체 드롭다운.

## Task 2: 카드 재배열 (▲▼)

**Files:** `web/src/app/editor/[jobId]/page.tsx`, `web/src/components/editor/LeftPanel.tsx`

- [ ] **Step 1: 에디터 페이지에 handleReorderCard 추가**

`editor/[jobId]/page.tsx`의 `handleImageUpdate` 콜백 **다음**에 추가:
```tsx
  const handleReorderCard = useCallback((idx: number, dir: -1 | 1) => {
    const target = idx + dir
    setLocalData((prev) => {
      const base = prev ?? apiData?.cardData
      if (!base) return prev
      if (target < 0 || target >= base.cards.length) return prev
      const next = [...base.cards]
      const tmp = next[idx]; next[idx] = next[target]; next[target] = tmp
      const renumbered = next.map((c, i) => ({ ...c, card_num: i + 1 }))
      const updated = { ...base, cards: renumbered }
      debouncedSave(updated)
      return updated
    })
    setActiveCardIdx((cur) => {
      if (target < 0 || target >= cards.length) return cur
      return target   // 이동한 카드를 따라 선택 이동
    })
  }, [apiData, debouncedSave, cards.length])
```

(d) ThumbItem 내부의 최상위 `<button type="button" onClick={onClick} ...>` 를 클릭 가능한 `<div>`로 전환(버튼 중첩 방지). 기존 button의 style은 유지하고 div로 바꾼 뒤 role/tabIndex/키보드 핸들러 추가. 그리고 썸네일 박스 우상단(번호 배지 반대편)에 ▲▼ 컨트롤을 absolute로 얹는다. 아래로 교체:

기존:
```tsx
      <button
        type="button"
        onClick={onClick}
        aria-label={`카드 ${idx + 1}`}
        style={{
          width: '100%', padding: 0, background: 'transparent', border: 'none',
          cursor: 'pointer', display: 'flex', flexDirection: 'column', gap: 6,
        }}
      >
```
→
```tsx
      <div
        role="button"
        tabIndex={0}
        onClick={onClick}
        onKeyDown={(e) => { if (e.key === 'Enter' || e.key === ' ') { e.preventDefault(); onClick() } }}
        aria-label={`카드 ${idx + 1}`}
        style={{
          width: '100%', padding: 0, background: 'transparent', border: 'none',
          cursor: 'pointer', display: 'flex', flexDirection: 'column', gap: 6,
        }}
      >
```
그리고 이 블록의 닫는 `</button>`를 `</div>`로 바꾼다.

(e) 썸네일 박스(`<div style={{ position: 'relative', width: thumbSize, ... }}>`) 안, "카드 번호 (top-left)" 블록 **다음**에 ▲▼ 컨트롤 추가(`onReorderCard`가 있을 때만):
```tsx
          {onReorderCard && (
            <div style={{ position: 'absolute', bottom: 6, right: 6, display: 'flex', flexDirection: 'column', gap: 4 }}>
              <button
                type="button"
                aria-label="위로 이동"
                disabled={idx === 0}
                onClick={(e) => { e.stopPropagation(); onReorderCard(idx, -1) }}
                style={{
                  width: 22, height: 22, borderRadius: 6, border: 'none',
                  background: 'rgba(255,255,255,0.92)', color: 'var(--ink)',
                  cursor: idx === 0 ? 'not-allowed' : 'pointer', opacity: idx === 0 ? 0.35 : 1,
                  display: 'flex', alignItems: 'center', justifyContent: 'center',
                  boxShadow: '0 1px 3px rgba(0,0,0,0.18)',
                }}
              >
                <span className="material-symbols-outlined" style={{ fontSize: 16 }}>keyboard_arrow_up</span>
              </button>
              <button
                type="button"
                aria-label="아래로 이동"
                disabled={idx === total - 1}
                onClick={(e) => { e.stopPropagation(); onReorderCard(idx, 1) }}
                style={{
                  width: 22, height: 22, borderRadius: 6, border: 'none',
                  background: 'rgba(255,255,255,0.92)', color: 'var(--ink)',
                  cursor: idx === total - 1 ? 'not-allowed' : 'pointer', opacity: idx === total - 1 ? 0.35 : 1,
                  display: 'flex', alignItems: 'center', justifyContent: 'center',
                  boxShadow: '0 1px 3px rgba(0,0,0,0.18)',
                }}
              >
                <span className="material-symbols-outlined" style={{ fontSize: 16 }}>keyboard_arrow_down</span>
              </button>
            </div>
          )}
```
